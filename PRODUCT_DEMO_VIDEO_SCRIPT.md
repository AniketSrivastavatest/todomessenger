# TodoMessenger Product Demo Video

## Video Goal

Create a 60-75 second product demo for STATION F, early customers, and investor conversations.

Positioning:

> TodoMessenger turns workplace conversations into assigned, trackable tasks with AI.

## Format

- Length: 60-75 seconds.
- Style: clean B2B SaaS demo.
- Tone: practical, confident, modern.
- Visual style: white UI, black text, cyan/blue accents, minimal motion.
- Output target: Canva video, website hero, LinkedIn post, STATION F application link.

## Main Message

Teams lose work because tasks start inside conversations. TodoMessenger makes those conversations actionable by default.
